# Installation-Linux-Termux

```
apt-get update && apt-get upgrade && apt-get install git python3 nodejs
```
```
git clone https://github.com/SudoR2spr/WDZONE-CC
```
```
cd WDZONE-CC
```
```
pip3 install -r requirements.txt
```
```
python3 cc.py
```
